方法论与方法说明

全球养老金政策动态追踪（Global Pension Policy Monitor）是一个专注于跟踪全球主要经济体养老金政策动态的研究项目。本网站基于公开信息源，系统性地收集、整理和展示各国养老金领域的最新发展。

数据来源

信息来源于以下类型的公开渠道：

  • 官方政府网站：各国养老金管理机构、劳动部门、财政部门的官方新闻发布
  • 国际组织：OECD、世界银行、国际劳工组织（ILO）的养老金研究报告和数据库
  • 监管机构：养老金监管局、金融监管机构发布的政策通告和监管更新
  • 新闻媒体：主要财经媒体的养老金相关报道（作为补充来源）

每日更新指南

如需添加新的动态条目，请按以下步骤操作：

  1. 定位数据文件：打开 pension-tracker/data/entries.json
  2. 新增条目：在 "entries" 数组中添加新的条目对象，格式如下： 
    {
  "id": 25,
  "country": "美国",
  "title": "条目标题",
  "content": "详细内容描述...",
  "date": "2026-04-24",
  "category": "政策调整",
  "importance": 3,
  "source": "来源机构名称",
  "url": "https://source-url.com"
}
  3. 更新元数据：修改 metadata.last_updated、metadata.total_entries 和报告周期
  4. 保存并刷新：保存 JSON 文件，刷新网站页面即可看到更新

提示：新增条目的 id 应为当前最大 ID 加 1，确保唯一性。
